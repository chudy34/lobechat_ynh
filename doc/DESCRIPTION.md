LobeHub is a full-stack, self-hosted AI workspace.

**Features:**
- 50+ LLM providers (OpenAI, Claude, Gemini, DeepSeek, Ollama, …)
- Knowledge Base with RAG (file upload, semantic search)
- MCP plugin support
- Multi-agent collaboration
- Built-in web search (SearXNG)
- File storage (RustFS S3)
- Multi-user with NextAuth

**Stack:** LobeHub + ParadeDB (PostgreSQL 17 + pgvector) + RustFS + Redis + SearXNG
